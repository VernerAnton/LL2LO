# Assets Folder

## Logo File

Place the company logo here as `suorahaku-logo.png` (or `.jpg`).

**Required:**
- File name: `suorahaku-logo.png` (or `suorahaku-logo.jpg`)
- Recommended size: 200-300px width
- Format: PNG (transparent background preferred) or JPG
- Location: `public/assets/suorahaku-logo.png`

This logo will be automatically added to the top-right corner of each generated slide.

## Extracting Logo from Template PPTX

1. Open your template PPTX file
2. Right-click on the logo image
3. Select "Save as Picture..."
4. Save as `suorahaku-logo.png`
5. Move to this folder
